# 集成学习

## 定义

通过组合多个模型的预测结果，提高整体性能的方法，如随机森林、梯度提升树等。

## 生活例子

就像团队合作，多个专家一起讨论，做出比单个专家更好的决策。

## 核心概念关联

- 结合多个 [[模型参数|模型]] 的预测
- 可以减少 [[过拟合]] 和 [[泛化能力|方差]]
- 典型方法：Bagging、Boosting

## 常见算法

| 算法 | 原理 |
|------|------|
| 随机森林 | Bagging + 决策树 |
| 梯度提升树 | Boosting + 决策树 |
| XGBoost | 梯度提升的高效实现 |
| Stacking | 模型串联集成 |
