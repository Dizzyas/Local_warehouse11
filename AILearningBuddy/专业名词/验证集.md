# 验证集

## 定义

用于在训练过程中评估模型性能的数据集，用于调整超参数。

## 生活例子

就像模拟考试，用于评估学生的学习情况，但不用于最终评分。

## 核心概念关联

- 来自训练数据的划分（不同于 [[测试集]]）
- 用于 [[交叉验证]]
- 监控 [[过拟合]]
- 辅助 [[超参数]] 调优

## 数据划分比例

```
原始数据
├── 训练集 70-80%
├── 验证集 10-15% ← 训练过程中用
└── 测试集 10-15% ← 最终评估用
```
