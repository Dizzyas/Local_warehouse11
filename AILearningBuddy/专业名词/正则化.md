# 正则化

## 定义

一种防止过拟合的技术，通过在损失函数中添加惩罚项，限制模型参数的大小。

## 生活例子

就像老师要求学生的答案简洁明了，避免冗长的解释。

## 核心概念关联

- 用于防止 [[过拟合]]
- 提升 [[泛化能力]]
- 通过惩罚大参数实现

## 常见方法

| 方法 | 说明 |
|------|------|
| L1正则化 | 产生稀疏参数（特征选择） |
| L2正则化 | 惩罚大参数（权重衰减） |
| Dropout | 随机丢弃神经元 |
| Early Stopping | 早停防止过拟合 |
