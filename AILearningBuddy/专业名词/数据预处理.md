# 数据预处理

## 定义

对原始数据进行清洗、转换和标准化的过程，使其适合模型训练。

## 生活例子

就像洗衣服，先将脏衣服分类、浸泡、清洗，使其干净整洁。

## 核心概念关联

- 是机器学习的第一步
- 处理内容包括 [[特征]] 清洗、缺失值、异常值
- 是 [[特征工程]] 的前置步骤

## 常见操作

| 操作 | 说明 |
|------|------|
| 缺失值处理 | 填充或删除 |
| 异常值处理 | 裁剪或替换 |
| 数据标准化 | 归一化到统一范围 |
| 类别编码 | 将类别转为数值 |
