# 神经网络

## 定义

一种模仿人脑神经元结构的机器学习模型，由输入层、隐藏层和输出层组成。

## 生活例子

就像一个团队，输入层接收信息，隐藏层的成员处理信息，输出层给出最终结果。

## 核心概念关联

- 是 [[深度学习]] 的基础
- 核心组件：[[激活函数]]
- 通过 [[反向传播]] 训练
- 常见架构：[[卷积神经网络]]、[[循环神经网络]]

## 网络结构

```
输入层 → 隐藏层1 → 隐藏层2 → ... → 输出层
```
