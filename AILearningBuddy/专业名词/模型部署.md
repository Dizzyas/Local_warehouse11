# 模型部署

## 定义

将训练好的模型集成到实际应用中的过程，使其能够处理新的数据并做出预测。

## 生活例子

就像将实验室开发的新药投入市场，让更多人受益。

## 核心概念关联

- 是机器学习项目的最终环节
- 需要考虑性能优化
- 常见方式：API服务、边缘部署、嵌入式

## 部署方式

| 方式 | 说明 |
|------|------|
| REST API | 通过HTTP接口调用 |
| 边缘部署 | 在设备端本地运行 |
| 批量预测 | 离线批量处理数据 |
