# 文字动画

## 文字基础认识
- Windows/Fonts字体文件夹去添加字体
- 文字动画音乐要求卡点
- 整体的动画留在最后去做,用空对象父子级
- 推荐镜头一个一个做,再到pr里面拼接
- 重要的工程要备份

### 字符面板
1. 修改字体,字体大小,行间距,字间距
	- 字符间距拉满了,还是觉得近,可以到文字图层>文本动画>字符间距:可以做到无限拉长
2. 填充色和描边色
3. 在填充上描边:描边在填充上
4. 在描边上填充:填充在描边上
5. 全填充,全描边
6. 无描边或填充:选中颜色方框,保持在上面,点左下角的斜杠方格
### 文字动画
- 添加了文字动画之后有一个范围选择器,在文字图层上显示为两个竖线,被扩上的区域才会被文字动画影响
- 缩放动画:在文本图层>更多选项>分组对齐里,可以调整缩放的锚点
- 不透明度动画:调整平滑度为0,就没有过渡动画;随机出现调整高级>随机排序
## 文字动画案例解析
![[06.文字动画案例.mp4]]
1. 文字逐帧缩放显示
	- 初始排版:文本图层添加动画>字符间距,拉长字符间
2. 文字图层>文本>缩放:每个字符的下方都有一个红叉号=锚点,字符按独立锚点缩放.在字符的开头和结束位置还有一个三角形符号,标记缩放的起始位置和结束位置.缩放实质是三角形范围里的缩放.
	- 缩放的文字的动画选择工具属性>范围选择器>起始,结束,偏移
	- k起始的帧,实现逐帧逐字缩放动画.起始k帧实质是调整三角形的动态范围,范围逐渐变大,文字的缩放属性就逐渐被影响
	- 缩放影响独立字符的缩放,起始影响要被缩放的文本范围
	- 缩放不用k帧,缩放是整体的,但是和图层的s缩放也不同.缩放的锚点是字符的下方
	- 调整缩放动画的锚点:文本图层>文本>更多选项>分组对齐>调整x,y,50%到字符中间
	- 细节:添加动画要分开,不要选择了一个动画制作工具后再添加动画;如果加在了一起,k了缩放起始的帧,它还会收到字符间距的影响,效果也不错.只要缩放动画就分开,需要字符间距和缩放一起的动画就添加在一起
3. 文字逐帧出现,没有缩放动画
	- 添加动画:文本图层>文本>添加动画>不透明度,范围选择起>起始k帧
	- 解决出现的半透明:范围选择器>高级>平滑度=0
	- 范围选择起>高级>依据>词,以空格区分;行;字符,不包含空格的字符
	- 高级>单位>索引>数值:以数字表示,默认百分比挺好的不用改
	- 高级>模式:相加,默认模式;相减:把初始和结束的纸做差,可以用来做收尾一起移动的动画;
4. 底下一排小圆点整体放大的动画
	- 添加缩放动画去做,按上面的方法
	- 中继器法(文本图层添加不了中继器):椭圆工具,无描边,画小圆,添加中继器,变换>位置,调整间距,调整图层的缩放是围绕描点整体的缩放,中继器里的缩放是围绕中继器锚点的圆点逐渐放大和缩小,因此需要调整椭圆路径里的大小.中继器是围绕第一个克隆出来的,所以要调本体的椭圆路径
5. 很多排小圆点的出现消失,中间出现和消失部分一起移动的动画
	- 在第一排做了中继器的基础上,再加一个中继器,中继器的属性>位置,调整y轴数值,x轴为0.
	- 用矩形工具画形状遮罩,k遮罩的关键帧
6. 中间大文字往中间缩小且有些字符不透明度0-100的动画
	- ctrl+home先居中,字符大小和间距调高一点
	- 添加动画字符间距>调整字符间距,变化方式可能不是从中间开始,展开方向不对,需要调整段落对齐方向,改成居中,就是从中间往两边
	- 随机不透明度:高级选项随机排序,需要单独添加不透明度,不要调一起了,k起始的关键帧.
7. 全屏随机闪烁的字符
	- 文本图层s不透明度属性>alt不透明度加表达式:wiggle(15,150),wiggle翻译成振动,第一个参数是频率(每秒变化15次左右的不透明度参数),第二个参数是范围(从0开始到150范围内选取参数,透明度的参数最高是100).
	- 这个表达式可以加在位置,缩放,旋转等上.
8. 整体的缩放
	- 新建空对象,做缩放的关键帧,链接父子级.
9. 背景图层的颜色切换
	- 纯色图层添加效果:右键图层>效果>生成>填充,k颜色的关键帧,打上定格关键帧
10. 文字的颜色变换
	- 也用填充的方法去做,把背景的关键帧复制过来,改填充色就行了.关键帧复制要对准时间轴.
11. 上下位置变化,随机,有的上,有的下.
	- 文本图层>添加动画>位置>动画制作工具添加动画>选择器>摆动,随机摆动
	- 摆动选择器>摆动每秒=0
12. 图片扭曲收缩动画,向屏幕内凹,周围模糊,中间清晰
	- 添加素材:想要拖到时间轴的特定位置,可以直接拖图片到时间轴面板上,出现小方块提示送左键.往左面的面板拖就是铺面
	- 添加效果>右键图层>效果>扭曲>凸出>水平半径,垂直半径,凸出中心,凸出高度:给负值就是凹下去;锥形半径就是过度=蒙版羽化.下去锯齿,高清晰,低模糊
	- 凹下去太多可能会出现边缘的破洞,需要勾选固定边缘.
	- 往下凹的动画:水平,垂直半径1000,高度-2,锥形半径80.高度打关键帧
	- 边缘模糊动画:右键图层>效果>模糊和锐化>径向模糊,中心点,类型改为缩放,数量,消除锯齿低,很卡.k数量的关键帧
	- 复制关键帧到后面的图片素材就可以了,但是照片的大小不一致,凸出点,径向模糊的点需要调整
	- 关键帧的位置需要往前拖,做关键帧的错帧,让镜头动借动

## 摄像机使用
1. 修改摄像机的快捷键ctrl+shift+y
2. 切换成摄像机工具,移动视图
	- 左键是旋转视图,鼠标中键是上下左右平移视图,右键是远近推拉镜头.
	- 22版之后被拆分成了三个单独的工具,按住alt键就和22版之前一样了.
3. ![[06.摄像机视图.png]]
4. 用两个视图,一个是活动摄像机视图作为输出,一个是自定义视图.
5. 选中的视图会有4个蓝色的角标

## 焦距
- 焦距变化,左边最小,右边最大
- ![[06.焦距.png]]
- 焦距越小,形变越明显,类似于摄像机的广角.作用于三维图形,平面无影响.一般用30,50,80的焦距

## 单节点和双节点最主要的区别
- ![[06.单双节点的区别.png]]
1. 单节点
	- 没有目标线和目标点
	- 位置属性,移动时就是平移
2. 双节点
	- 有目标线和目标点
	- 位置属性,移动时围绕目标点移动
	- 可以调整目标点

## 摄像机动画
1. 在移动摄像机前要移动图层的z轴位置,不能直接对摄像机k帧
2. 移动图层时可能会出现穿帮,可以在移动后调整s缩放.
3. 不要摆的太平均,要有错落感.这样空间效果才好
4. 做摄像机的关键帧,边做边看有没有穿帮且调整位置.
5. 拉关键帧曲线时,可以在速度不够时,在两个关键帧中间再加一个关键帧,把这个帧往后移,做一个先快后慢的曲线.
6. 在摄像机选项里开景深开关,在焦距里的图层就是实的,和相机对焦一样.
## 其他动画
1. 云朵动画可以直接k帧
2. 城墙位置烟雾瀑布的动画.
	- 拆解成很多个圆,做缩放位置的关键帧
	- 用振动表达式:位置alt+wiggle(8,10)或者缩放
	- ctrl+D复制,调整缩放.
	- 把这些图形动画打个预合成加三维属性,往后移.
3. 文字动画
	- 文字添加位置属性
	- 调位置参数
	- k范围选择器帧
	- 高级>依据>词
	- 画蒙版或者遮罩,不动下面的总图层的位置属性,蒙版是不动的.
	- 最后下去的时候是从左到右的下去,位置动画高级模式>相减,k结束关键帧
4. 线条动画
	- 形状图层拉长
	- 添加修剪路径动画
	- 反转路径![[06.反转路径选项.png|329]]
5. 风车旋转动画
	- 先用锚点平移工具移动到锚点到中间
		- 正常的图层只用一个旋转轴,三维图层有三个旋转轴

		- 用方便做旋转的表达式,时间表达式:`time*90`(每秒转90°)
6. 补一个蓝色的纯色图层

# 三维和文字动画组合案例
## 摄像机和图层动画
1. 把图片都摆完,用三维场景神器,可以改焦距,排布,层间距
2. 可以用三维场景神器来排布,但是是按照图层的顺序来排的.
3. 处理雾气图层:白色的雾气图层变成绿蓝色,加三色调效果,高光,中间值,阴影吸色或者Lumetri>曲线:调颜色
4. 水波效果的实现:
    1. 做流动的贴图
        - 新建纯色图层,添加效果分形杂色,关掉变换里的统一分辨率,可以让原先的一坨变成分散的效果,接着把缩放高度改为10,把之前的压缩了,更符合水波的细碎.
        - 利用time表达式,添加到演化到,演化可以使杂色的效果产生左右的流动效果.
        - 这个贴图是不需要看的,需要关掉.
    2. 贴到水的图层上
        - 添加一个调整图层,放在水的图层上一层,加置换图的效果(识别调整图层下面的图层)
        - 置换图层是水贴图图层(纯色图层)和效果和蒙版,这样可以使水贴图的效果引用到水的图层上,使水流动起来.
        - 调整最大水平置换为100-150,凸显水平的细微流动
        - 把最大水平置换和最大垂直置换都改为明亮度,不需要蓝色和红色,让变换效果更自然
        - 对月亮和底图星空上有影响的话可以打预合成或者调整图层的上下关系.
    3. 其他的问题的补充
        - 问题:产生模糊和黑块.
        - 利用动态拼贴效果补充两边的不够的地方.
        - 动态拼贴:拼贴复制,把水波纹的两边没有出现的位置做一个拼贴.
        - 动态拼贴效果放在置换图的上面,避免产生黑边.
5. 近处石头的倒影:把图层复制一层,缩放属性的中间y轴取相反数(轴向翻转),添加线性擦除效果,调整过度完成的参数,擦除的角度,还要加羽化.
6. 虚实变幻效果的呈现:在摄像机下面的打开景深开关,调整焦距(有白色框),光圈调低一点.定焦作用
## 文字的动画
1. 文字图层,横排的也可以,调整行间距大一些.开启三维
2. 调整y轴的大小,可以利用三维插件去做

7. 添加cc light sweep效果:在贴图上做一个线条高光效果,做完再alpha遮罩
