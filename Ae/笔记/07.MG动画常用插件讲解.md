# MG动画制作流程
1. 概念:会运动的图形设计,融合了平面设计,动画设计和电影的表现语言.
2. 应用范围:电影电视的片头片尾,企业的产品以及形象宣传,科普传播.
3. MG动画和传统动画的区别:
    - 传统动画:需要有主线故事并通过塑造不同角色来讲好故事
    - MG动画:通过图形和文字的动态化变现传递思想或者情感

# 牛顿插件
### 在合成面板下,弹窗独立打开
1. 会有红字提示,需要破解,左上角牛顿注册一下
2. 多形状图层打开牛顿会提示你分离.
3. 不要显示背景,牛顿也不会显示图案.
## 面板
1. 左上角文件>打开自动设置,关掉牛顿插件会保存动力学设置,下次打开还在
2. 改背景颜色,无限地面,设置边框
![[09.无限地面.png]]
3. 样式(默认是动态):自由落体
	- 静态:不动
	- 休眠样式:碰到了才会动
	- AEmatic(牵引):被一个弹力绳拉着.空气阻力和空气张力可以调整拉扯速度和长度的
	- 死:不参与动画
	- 运动学:按我们在ae里k好的关键帧,参与到动力学里运动.
4. 速度工具,物体做抛物线运动
5. 导出:有红色的就注册,勾选新合成.图层是每帧的关键帧动画
![[09.速度工具.png|389]]
## 参数
1. 密度
2. 摩擦
3. 弹性
4. 网格精度:弧形的图形的圆润程度,开高了碰撞面变多,变卡.渲染没影响
5. 角速度
6. 角阻力:产生角速度的阻力
7. 重力:可调整方向
## 高级板块
1. 有针对性的碰撞,虽然在碰撞的路径上但是又互不干扰:高级属性里面的碰撞组.
2. 把需要针对的选择同一个组,把不在一个组里的组取消下面的碰撞
3. 固定旋转:在方块往下掉时会受到重力的影响,往旁边掉,在勾选固定旋转之后就不会往侧面掉落(类似搭箱子)
4. 重力缩放:单独图层物体受重力的影响,数值越大就掉的快,负值为向上的重力(气球)
5. 凸面外壳:把文字图层或者logo的图层封闭起来,上面掉落的形状掉落下来就之后落在这个图层的上边缘,不会从中间穿过去.
6. 磁性:给图层开启引力或者排斥力,有磁铁强度,范围参数.可以做吸尘的动画.其他物体要是不想去受到磁性影响可以不勾选接受磁性
7. 不建议用导出联系,后期不好调参数,它用的是滑块控制.
8. 解算器(左边的全局属性):物体与物体之间会有穿模,调整碰撞容积参数,给高了会使物体挨着了也不发生物体的碰撞,相隔多远去解析碰撞.
9. 子步骤:参数代表算力,越高越精细,高了卡
10. 分时器:越高速度越慢,做子弹的慢放效果.线性阻力是单个图层的,这个分时器是全局的.
## 作业-愤怒的小鸟
导入>图层-保持图片大小,ctrl+s保存. 
1. 地面和弹弓是死亡体
2. 小鸟和地面上的是休眠体,小鸟的密度要高一点
3. 用速度工具,拉小鸟的运动轨迹
4. 小鸟往后拉的动画:先用牛顿插件做完之后再去改关键帧.在第一个关键帧时往后拉一个后面的关键帧,还要加一个变速.
5. 弹弓的绳子动画制作
- 用钢笔工具做曲线,关闭填充,但是要让小鸟夹在这两根线中间.
- 打钢笔图层路径关键帧,先拉伸后边直再变回来会弹的动画.
- 路径动画做不了弹性表达式.
- 拆分为两根,复制画出来的橡皮筋,先隐藏一个,点应该在上面一根一个点删除,下面的也是把不在上面的一个点删掉.修剪路径也可以做.
- 把小鸟的动画做了之后再去做橡皮筋的动画.
