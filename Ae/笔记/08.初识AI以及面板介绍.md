# 创建
1. ctrl+N新建文档,选web-1920x1080,左边命名
2. 单位:像素;方向:横版竖版,画板1,一个镜头一个画板.
3. 出血不用管除了打印
4. 高级选项默认
5. 改传统基础功能/ctrl+s保存
# 基础操作
1. 导入工程:直接拖将来,拖到边框就是新建一个工程并打开
2. alt+滚轮放大缩小
3. 匹配图层大小:鼠标放到四角缩放图片,按住shift不变比例.
4. 到正中间:多参考线
5. 锁定:ctrl+2;解锁:ctrl+alt+2或者图层下面解锁图标
6. 改颜色:吸管工具
7. 上边栏改描边,填充.
8. 复制图层:ctrl+c,ctrl+v;ctrl+c,ctrl+f原地复制;alt拖,有位置对位显示.
9. ctrl+d:复制上一步的动作.
10. 选中描边,可以用滚轮调节数值.
11. 钢笔绘制路径:左键按住就是空间贝塞尔,shift就是直线.
12. 吸管工具会把描边和填充都洗过来.不考虑:shift洗,描边或者填充要在上面.
13. 描边下边可以下拉打开选择内描边.
14. 对不起:窗口关闭像素对齐
15. 方向:鼠标放在边角看图标变了就可以转了,按住shift就是45°转
# 案例1
只有图形
1. 多边形工具,按住shift正多边形,上下键控制边数.
2. ctrl+d复制上一个
3. 布尔运算:窗口里面找,形状模式有交集,差集,并集;分割:在编组里看拆的图层,和预合成相似,用小白工具可以选择编组,小标工具也可以改图形的某个点的位置来改形状,还可以改角的圆度,点小白点.小黑点改的是所有角的圆度.
4. 一条直线变成扭曲效果:效果面板>扭曲和变换>用波纹效果,开启预览,平滑.秒变改成圆头

# 案例2
有透视关系,渐变,假三维,文字
1. 渐变:左边填充颜色下>渐变,径向线性渐变>拉中间的线改位置和效果,双击颜色条吸颜色.
2. 三维:先画一个三角形,alt复制一份;假三维颜色不用,要连在一起,用小白工具,把一个点吸附到另一个点.
3. 文字:文字工具里横排和竖排区域文字路径文字,右边改字符,字体
4. 不规则形状:用钢笔绘制一个大概,再用小白工具调整圆度
5. 线条:钢笔画
6. 图标:先做一个,再点右边梅花>新建>选择图标>喷枪,还有位置喷枪工具,还用缩放喷枪工具,按alt键就是反向.还可以导入其他素材.
