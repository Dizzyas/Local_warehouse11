# **一、操作快捷键**

撤回：Ctrl+Z 
保存：Ctrl+S 
复制：Ctrl+D 
裁剪：Ctrl+Shift+D 
渲染合成：Ctrl+M 
清理内存：Ctrl+Alt+/ 
导入素材：Ctrl+I 
把纯色铺满整个合成画面:ctrl+alt+F
修改摄像机和纯色图层:ctrl+shift+Y
启用时间重映射:ctrl+alt+T
删除所选图层所有效果：Ctrl+Shift+E 
新建纯色图层：Ctrl+Y 
新建调整图层：Ctrl+Alt+Y 
新建文字图层：Ctrl+Shift+T 
关键帧添加缓动：F9 
关键帧添加缓入：Shift+F9 
快速选择多个图层：选中第一个图层然后按住Shift选择第N个图层 
编辑关键帧速率：Ctrl+Shift+K 
文字工具：Ctrl+T 
新建合成：Ctrl+N 
根据所选图层创建预合成：Ctrl+Shift+C 
修改合成设置：Ctrl+K 
选择工具：V 
抓手工具：H 
缩放工具：Z 
旋转工具：W 
摄像机：C 
锚点工具：Y 
形状工具：Q 
钢笔工具：G 
文字工具：CMD+T 
画笔工具：CMD+B 
人偶控制：CMD+P 
临时抓手：空格 
缩放工具：按滚轮进行缩放

# **二、剪辑快捷键**

前进/后退1帧：Ctrl+→/← 
前进/后退10帧：Shift+→/← 
跳到图层入点：I 
跳到图层出点：O 
设置图层入点：[
设置图层出点：] 
删除当前时间之前：Alt+[
删除当前时间之后：Alt+] 
设定工作区域起始：B
设定工作区域结束：N 
预览合成：空格 
给图层添加标记：Ctrl+8

旋转：R 
缩放：S 
关键帧值：U 
位置：P
