# 图生图
1. 按照原先的图尽量描述
2. 降低重绘幅度,固定种子

# 局部重绘
1. alt加滚轮缩放画布
2. ctrl加滚轮调整笔刷
3. R重置缩放
4. S全屏模式
5. F移动画布
6. 可以把人物和背景分开(黑白)
7. Ae把视频转成png,批处理

# 批处理
1. 到Ae里新建合成
2. 不要切换景别
3. 改目标区域
4. ctrl+m,输出格式,png序列
5. SD批量处理输入,输入目录输出目录

# mov2mov
1. SD插件安装,SD插件需要复制到extension里,要重启,装模型时可以不重启
2. 把视频拖到mov里面
3. controlnet:加预处理器.使图片生成更像,只能识别图片
    1. 启用控制单元,最多三个控制单元.
    2. 线稿或深度(两个控制单元)
    3. 自动设置预处理器分辨率,开启预览,点红色的爆炸按钮
    4. 使用独立的控制图像
    5. 控制接入时机就是:ai自己生成一些,后面按图生成
    6. 预览越分明越好

# 序列控制
1. 素材准备:明亮深度图,局部重绘图,被扣出来
2. 调色处理:右键>效果>颜色校正>曲线
3. 背景右键>效果>扭曲>置换图,利用深度图置换图层,做水平竖直的拉伸变换,明度置换,在数值比较大时,两边会穿帮
4. 右键>效果>风格化>cc repotile(可再生的),动态拼贴.动态拼贴只能在合成里面拼贴,不能在外面延长,但是cc repotile可以往外面复制,可以调整复制模式.
5. 图层顺序:cc repotile在置换图上面
6. 月亮的辉光:新建调整图层加deep glow,F蒙版羽化
7. 汽车的车灯(有模糊,光斑):新建调整图层>效果>cc light burst和deep glow,链接父子级,使光路匹配车的移动.
8. 新建纯色>右键>效果>video copilot>OF光,点进option里,渲染模式换成透明,颜色自己调,T键调整不透明度
9. 尾气制作:类似波形变形,新建纯色>用钢笔工具绘制大致烟雾形状,添加波纹变形,调整宽度和高度,固定位右边,速度-1.ctrl+D复制一份,调整宽度高度等参数.
10. ctrl+D复制图层,叠加选择模式轮廓α.![[14.SD 车动画.mp4]]
