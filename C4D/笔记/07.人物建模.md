# 躯干
1. 躯干:FFD+圆柱,加了FFD效果更好
2. kk:手臂位置,单切,米字形
3. ms倒角:八边形
4. 面拉出来一点,缩放里变成扁平,然后删掉
5. 删掉一半,用对称(打开焊接对象)
6. 底部两个细分面桥接,KL先k了之后在长条里点加号
7. 圆柱上部分KL做脖子:删掉中间,然后往上延伸
8. Q控制启用生成器
# 四肢
1. 四肢:圆柱体,改为8细分,无封顶
2. 脖子同理,分段是12
3. 四肢与主体的连接用桥接
4. 在连接四肢前尽量把中间做好,以后改不好选点
5. 开细分曲面选点改细节,手臂,肩膀,斜方,肚子,大腿,腹部,腰部
6. 手指:本体正方体,拉成长方体,挤压(关掉保持群组),手指细分拉三节,大拇指在旁边,旋转,拉伸,各种调整
7. 脚: 选面挤出,打平,
